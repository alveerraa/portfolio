## 📝 Setup Steps

### 1. Build the Website
- `index.html` – Homepage
- `style.css` – Styling
- `script.js` – Interactivity (if any)

### 2. Upload to S3
- Create an S3 bucket
- Enable static website hosting
- Upload your files
- Set `index.html` as the index document
- Make files public (if required)
- Access your site via the public URL
